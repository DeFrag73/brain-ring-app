# Brain Ring 🧠

## Як запустити

### Крок 1 — Встановити Docker Desktop
Завантажте та встановіть з офіційного сайту:
👉 https://www.docker.com/products/docker-desktop/

Під час встановлення приймайте всі налаштування за замовчуванням.
Після встановлення перезавантажте комп'ютер.

### Крок 2 — Запустити додаток
1. Відкрийте термінал (або PowerShell на Windows) у папці проєкту.
2. Виконайте команду:

   ```
   docker compose up --build
   ```

3. Зачекайте, поки з'явиться повідомлення:
   `Uvicorn running on http://0.0.0.0:8000`

### Крок 3 — Відкрити в браузері
Перейдіть за адресою: **http://localhost:8000**
