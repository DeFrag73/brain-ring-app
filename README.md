# 🧠 Брейн-ринг - Інструкція по встановленню

## 📋 Передумови

Перед встановленням переконайтеся, що у вас є:
- Python 3.8 або новіша версія
- Git (опціонально, для клонування репозиторію)

## 🚀 Крок 1: Створення проєкту

### На Windows:
```cmd
# Створіть папку для проєкту
mkdir brainring-app
cd brainring-app

# Створіть віртуальне середовище
python -m venv venv

# Активуйте віртуальне середовище
venv\Scripts\activate
```

### На Mac/Linux:
```bash
# Створіть папку для проєкту
mkdir brainring-app
cd brainring-app

# Створіть віртуальне середовище
python3 -m venv venv

# Активуйте віртуальне середовище
source venv/bin/activate
```

## 📁 Крок 2: Структура файлів

Створіть наступну структуру папок і файлів:

```
brainring-app/
├── main.py
├── models.py
├── database.py
├── requirements.txt
├── templates/
│   ├── admin.html
│   └── display.html
├── static/
│   ├── style.css
│   └── script.js
└── brainring.db (створюється автоматично)
```

### Створення папок:
```bash
# У корені проєкту виконайте:
mkdir templates
mkdir static
```

## 💾 Крок 3: Створення файлів

Скопіюйте код з артефактів у відповідні файли:

1. **main.py** - основний файл додатку
2. **models.py** - моделі бази даних
3. **database.py** - налаштування бази даних
4. **requirements.txt** - список залежностей
5. **templates/admin.html** - адміністративна панель
6. **templates/display.html** - екран глядачів
7. **static/style.css** - стилі
8. **static/script.js** - JavaScript

## 📦 Крок 4: Встановлення залежностей

```bash
# Встановіть всі необхідні бібліотеки
pip install -r requirements.txt
```

Якщо виникають проблеми, встановіть по одній:
```bash
pip install fastapi
pip install uvicorn[standard]
pip install sqlalchemy
pip install python-multipart
pip install jinja2
```

## 🗄️ Крок 5: Ініціалізація бази даних

```bash
# Запустіть ініціалізацію бази даних з тестовими даними
python database.py
```

Ця команда:
- Створить файл бази даних `brainring.db`
- Створить всі необхідні таблиці
- Додасть приклади питань і команд

## ▶️ Крок 6: Запуск додатку

```bash
# Запустіть сервер
python main.py
```

Або альтернативно:
```bash
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

## 🌐 Крок 7: Використання

Після запуску додаток буде доступний за адресами:

- **Адмін панель**: http://localhost:8000/admin
  - Логін: `admin`
  - Пароль: `brainring2025`
  
- **Екран глядачів**: http://localhost:8000/display

## 🎮 Використання додатку

### Адміністративна панель:

1. **Питання**:
   - Додавайте нові питання
   - Редагуйте існуючі
   - Позначайте як використані
   - Показуйте на екрані глядачів

2. **Команди**:
   - Створюйте команди
   - Вказуйте склад і капітана
   - Видаляйте ненужні команди

3. **Поточна гра**:
   - Обирайте команди для раунду
   - Керуйте рахунком (+1, +0.5, -1, -0.5)
   - Завершуйте раунди
   - Автоматично записуйте результати

4. **Статистика**:
   - Переглядайте результати команд
   - Аналізуйте кількість перемог
   - Відстежуйте прогрес турніру

### Екран глядачів:
- Автоматично показує поточні команди
- Відображає рахунок у реальному часі
- Показує питання коли адмін їх публікує
- Красивий, професійний дизайн для проєктора

## 🔧 Налаштування

### Зміна паролю адміна:
У файлі `main.py` знайдіть і змініть:
```python
ADMIN_USERNAME = "admin"
ADMIN_PASSWORD = "your_new_password"
```

### Підключення PostgreSQL:
1. Встановіть залежність:
   ```bash
   pip install psycopg2-binary
   ```

2. У файлі `database.py` змініть:
   ```python
   DATABASE_URL = "postgresql://username:password@localhost/brainring"
   ```

## 🚨 Можливі проблеми та рішення

### Проблема: "Module not found"
**Рішення**: Переконайтеся що віртуальне середовище активоване і всі залежності встановлені.

### Проблема: "Port already in use"
**Рішення**: Змініть порт:
```bash
python -c "import main; import uvicorn; uvicorn.run(main.app, host='0.0.0.0', port=8080)"
```

### Проблема: База даних не створюється
**Рішення**: Запустіть `python database.py` перед запуском основного додатку.

### Проблема: Статичні файли не завантажуються  
**Рішення**: Переконайтеся що папки `templates` і `static` створені і містять відповідні файли.

## 🌟 Додаткові можливості

### Автозапуск при включенні комп'ютера (Windows):
1. Створіть файл `start_brainring.bat`:
   ```batch
   @echo off
   cd /d "C:\path\to\your\brainring-app"
   call venv\Scripts\activate
   python main.py
   ```

2. Додайте цей файл у автозавантаження Windows.

### Запуск на різних пристроях:
Якщо потрібно щоб додаток був доступний з інших комп'ютерів у мережі:
```bash
python main.py --host 0.0.0.0
```
Тоді додаток буде доступний за IP адресою вашого комп'ютера.

## 📞 Підтримка

Якщо виникають проблеми:
1. Перевірте що всі файли створені правильно
2. Перевірте що віртуальне середовище активоване
3. Перевірте логи в консолі для помилок
4. Переконайтеся що порт 8000 не зайнятий іншими програмами

## 🔄 Оновлення

Для оновлення додатку:
1. Створіть резервну копію файлу `brainring.db`
2. Замініть файли коду новими версіями
3. Перезапустіть сервер
